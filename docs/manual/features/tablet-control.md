# Tablet: O Controle Remoto da Reunião

O Tablet é a interface ativa do sistema. Tudo que você toca no Tablet acontece instantaneamente no Video Wall.

## 🎮 Navegação Básica
- **Seleção de Projeto**: Toque em qualquer card de projeto para atualizar as 4 telas do Wall.
- **Troca de Tela**: Use o seletor de tela (1-4) para focar uma das telas do Wall em tela cheia no Tablet (espelhamento).

## 🔍 Filtros de Precisão
No topo do Tablet, você encontrará filtros por:
- **Programa**: Agrupe projetos por tipo (ex: BESS, Subestação, Linha de Transmissão).
- **Status**: Filtre por em andamento, pausado ou concluído.
- **Busca por Voz**: (Opcional) Integre comandos de voz para encontrar projetos rapidamente.

## 📊 Insights de IA
Ao selecionar um projeto, toque no ícone de "Cérebro" para:
1.  **Risk Audit**: Ver o relatório de riscos proativos gerado pela IA.
2.  **Executive Brief**: Gerar um texto pronto para ser copiado para o e-mail semanal.

---

> **Dica**: O Tablet funciona melhor em dispositivos Surface ou iPads pro, mas o link é acessível via qualquer navegador moderno com suporte a WebSockets.
