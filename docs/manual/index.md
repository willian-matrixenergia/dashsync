# Central de Ajuda SyncDash

Bem-vindo ao manual oficial do **SyncDash**. Esta plataforma foi projetada para transformar a gestão de portfólios de infraestrutura através de sincronização em tempo real entre **Video Walls** e controles remotos via **Tablets**.

---

## 🚀 Por onde começar?

Dependendo do seu papel no projeto, escolha um dos guias abaixo:

### 👥 Perfis de Usuário
*   **[Executivo / Stakeholder](./personas/executivo.md)**: "Quero ver a saúde do meu portfólio no Video Wall."
*   **[Coordenador de Obra / Dados](./personas/campo.md)**: "Preciso atualizar os dados de avanço e mídias das obras."
*   **[Equipe de TI / Admin](./personas/t-i.md)**: "Preciso configurar o servidor, banco de dados e chaves de API."
