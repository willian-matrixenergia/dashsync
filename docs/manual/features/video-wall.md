# Screen 01–04: O Video Wall em Detalhes

O Video Wall é a interface passiva do SyncDash, otimizada para visualização a longa distância em salas de reunião.

## 📱 Screen 01: Portfolio Dashboard
- **KPI Cards**: Orçamento total, homens-hora (HH) e dias sem acidentes.
- **Tabela de Criticidade**: Lista dinâmica de projetos. Use o Tablet para filtrar por criticidade (Vermelho no topo).
- **Gantt Chart**: Visualização de cronograma macro para o projeto selecionado.

## 📈 Screen 02: Curva S e Produtividade
- **Gráfico Curva S**: Compara o percentual planejado (acumulado) vs. realizado.
- **Histograma de Mão de Obra**: Mostra o contingente previsto vs. real no canteiro.
- **Velocímetros**: KPIs de produtividade por disciplina.

## 🖼️ Screen 03: Galeria de Mídias
- **Lightbox**: Quando o tablet seleciona uma foto, ela expande nesta tela.
- **Tours 360°**: Navegação imersiva dentro do projeto.
- **Timelapses**: Vídeos curtos compilando semanas de trabalho.

## 📹 Screen 04: Monitoramento ao Vivo
- **Proxy de Câmera**: Fluxo de vídeo direto do canteiro.
- **Segurança**: O stream é transmitido via servidor do SyncDash para ocultar as credenciais originais das câmeras da rede externa.

---

> **Importante**: Se as telas ficarem pretas, verifique a conexão do servidor com a pasta de mídias (`/media`).
