# Quick Start Guide: SyncDash

Este guia ajudará você a ter o SyncDash funcionando em menos de 5 minutos.

## 1. O "Momento Aha!"
O SyncDash brilha quando você altera um dado em uma planilha Excel e vê o gráfico no Video Wall ser atualizado **instantaneamente**, sem precisar dar F5.

## 2. Passo a Passo Inicial

### Para o Coordenador de Dados
1.  Acesse a pasta `./data` no servidor.
2.  Abra o arquivo `Base01.xlsx`.
3.  Altere o status de criticidade de um projeto.
4.  Salve o arquivo.
5.  **Resultado**: O Video Wall emitirá um sinal de atualização e mostrará o novo status.

### Para o Controlador da Reunião
1.  Acesse a URL do Tablet (ex: `http://tablet.sync-dash.local`).
2.  Toque em um projeto no Tablet.
3.  **Resultado**: As 4 telas do Video Wall carregarão simultaneamente os dados, fotos e câmeras daquele projeto.

## 3. Atalhos Rápidos
- **Shift + R**: Força o recarregamento dos dados no Video Wall.
- **Clique Longo no Tablet**: Abre o sumário executivo gerado por IA.

---

> **Dica**: Sempre mantenha as planilhas fechadas após editar para evitar conflitos de bloqueio de arquivo pelo Excel.
